# 🎯 AI CCTV Surveillance System

An advanced real-time AI-powered CCTV surveillance system built using YOLOv8, ByteTrack, OpenCV, and Streamlit.

This project performs intelligent video surveillance with real-time people detection, tracking, crowd monitoring, intrusion detection, and live analytics dashboard.

---

## 🚀 Features

- Real-time People Detection using YOLOv8  
- Multi-Object Tracking (ByteTrack)  
- Entry & Exit Counting System  
- Crowd Detection & Alerts  
- Intrusion Detection in Restricted Area  
- Live CCTV Feed Dashboard  
- Real-Time Analytics Charts  
- Session Statistics & Logging  
- Beautiful Dark Theme UI (Streamlit + Plotly)

---

## 📷 System Features

### ✔ Live Video Processing
Processes webcam or video file in real time.

### ✔ Smart Tracking
Each person is assigned a unique ID using ByteTrack.

### ✔ Entry / Exit Counting
Tracks movement across a virtual detection line.

### ✔ Crowd Alerts
Triggers warning when people exceed threshold.

### ✔ Analytics Dashboard
Displays:
- People count over time
- Entry vs Exit graph
- Traffic distribution
- Session summary table
